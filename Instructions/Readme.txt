Deployment : 
Have xampp running with Apache and Mysql
Create an empty database called 'herbalife_db' for the project in mysql (phpmyadmin).
Copy only code folder to htdocs in xampp and rename code folder to any context name you want. Edit the wp-config.php file and change the db username and password to your local mysql db credentials.
Open in browser : localhost/<foldername>/index.php
Follow instructions on wordpress
